**SYSTEM_PROMPT constant (per D-06, D-07, D-09, D-13, D-14):**
```javascript
const SYSTEM_PROMPT = `You are a LinkedIn content creator writing automation portfolio posts for a home service company owner.

STORYTELLING FRAMEWORK: Before-After-Bridge (BAB)

- Frame everything as "home service company" — never mention painting, painters, or any specific trade.

TOOL NAMING RULES (STRICT):

  - "your calendar" (never Google Calendar)
  - "your email" (never Gmail)
  - "your spreadsheet" (never Google Sheets)
  - "accounting software" (never QuickBooks)
  - "time tracking app" (never QuickBooks Time)
  - "SMS platform" (never RingCentral)

HASHTAGS: End with exactly 3 hashtags on a new line:
1. #Automation (always)
2. One journey-stage tag (e.g., #LeadGeneration, #InvoiceAutomation, #TimeTracking, #ExpenseManagement, #ProjectManagement, #CRM, #Scheduling)
3. #Make, #Zapier, or #n8n (matching the platform)

FORMAT: Write the post as plain text suitable for LinkedIn. Use line breaks for readability. Do NOT use markdown headers, bold, italic, or bullet points — LinkedIn doesn't render markdown. Use emoji sparingly (max 2-3 per post). Use short paragraphs (2-3 sentences each).`;
```

**generatePost(bundle, platform) function:**

Parameters:
- `bundle`: bundle object from bundles.json
- `platform`: 'make' | 'zapier' | 'n8n'

Implementation:
1. Require `@anthropic-ai/sdk`: `const Anthropic = require('@anthropic-ai/sdk').default || require('@anthropic-ai/sdk');`
2. Create client: `const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY })`
3. Build user prompt from bundle data:
```
Write a LinkedIn post for this automation workflow:

Platform: ${platformName}  (use this platform name throughout the post)
Business Capability: ${bundle.title}
Journey Stage: ${bundle.journeyStage}

BEFORE (the pain your reader relates to):
${bundle.pain}

AFTER (the automated dream state):
${bundle.solution}

Key Automation Steps (reference these naturally in the Bridge section — don't list them as bullet points, weave them into the narrative):
${bundle.idealizedSteps.map((s, i) => `${i + 1}. ${s.label}`).join('\n')}

Known inefficiencies this automation fixes:
${bundle.inefficiencies.join(', ') || 'Manual processes, inconsistent timing, human error'}

Write the post now. Remember: 300-500 words, hook in first 210 chars, BAB framework, casual expert tone, end with mixed CTA + 3 hashtags.
```
4. Call Claude API:
```javascript
const message = await client.messages.create({
  model: 'claude-sonnet-4-20250514',
  max_tokens: 2000,
  temperature: 0.7,
  system: SYSTEM_PROMPT,
  messages: [{ role: 'user', content: userPrompt }]
});
```
Note: Use `claude-sonnet-4-20250514` (not opus) to manage API costs for 36 posts. Sonnet is excellent for creative writing at this length.
5. Extract text: `const postContent = message.content[0].text`
6. Return the post content string

**validatePost(content, platform, journeyStage) function:**

Returns `{ valid: boolean, errors: string[] }`:
- Word count between 300 and 550 (slight buffer above 500 for validation)
- First line (split by `\n`) is under 215 characters (slight buffer)
- Contains `#Automation` hashtag
- Contains platform hashtag (`#Make`, `#Zapier`, or `#n8n`)
- Does NOT contain forbidden tool names: 'PaintScout', 'CompanyCam', 'YouCanBook', 'RingCentral', 'QuickBooks Time', 'Integromat', 'Make.com' (case-insensitive check)
- Does NOT contain markdown formatting: no lines starting with `#` (except hashtags at end), no `**`, no `- ` bullet lists

**generateAllPosts(bundles, outputDir) function:**

1. For each bundle, for each platform ('make', 'zapier', 'n8n'):
   a. Call `generatePost(bundle, platform)`
   b. Call `validatePost(content, platform, bundle.journeyStage)`
   c. If validation fails, log warnings but still write the file (manual review will catch issues)
   d. Write to `path.join(outputDir, bundle.journeySlug, `${platform}.md`)`
   e. Log progress: `console.log('Generated: posts/${bundle.journeySlug}/${platform}.md (${wordCount} words, ${valid ? 'PASS' : 'WARN: ' + errors.join(', ')})')`
   f. Add 2-second delay between API calls to avoid rate limits: `await new Promise(r => setTimeout(r, 2000))`
2. Return summary: `{ total, passed, warnings, errors }`

```javascript
const fs = require('fs-extra');
const path = require('path');
const { generateAllPosts } = require('../lib/post-generator');

async function main() {
  if (!process.env.ANTHROPIC_API_KEY) {
    console.error('ERROR: ANTHROPIC_API_KEY environment variable not set');
    console.error('Get your key from: https://console.anthropic.com/');
    process.exit(1);
  }

  const dataDir = path.join(__dirname, '..', 'data');
  const postsDir = path.join(__dirname, '..', 'posts');

  const bundlesData = await fs.readJSON(path.join(dataDir, 'bundles.json'));
  console.log(`Generating posts for ${bundlesData.totalBundles} bundles x 3 platforms = ${bundlesData.totalBundles * 3} posts`);

  const result = await generateAllPosts(bundlesData.bundles, postsDir);

  console.log('\n--- Summary ---');
  console.log(`Total: ${result.total}`);
  console.log(`Passed validation: ${result.passed}`);
  console.log(`Warnings: ${result.warnings}`);
  if (result.errors > 0) {
    console.log(`Errors: ${result.errors}`);
  }
}

main().catch(err => { console.error(err); process.exit(1); });
```

**Script (`linkedin-posts/scripts/generate-diagrams.js`):**

```javascript
const fs = require('fs-extra');
const path = require('path');
const { renderAllDiagrams } = require('../lib/diagram-renderer');

async function main() {
  const dataDir = path.join(__dirname, '..', 'data');
  const postsDir = path.join(__dirname, '..', 'posts');

  const bundlesData = await fs.readJSON(path.join(dataDir, 'bundles.json'));
  console.log(`Generating diagrams for ${bundlesData.totalBundles} bundles x 3 platforms = ${bundlesData.totalBundles * 3} diagrams`);

  const results = await renderAllDiagrams(bundlesData.bundles, postsDir);

  const successes = results.filter(r => !r.error);
  const failures = results.filter(r => r.error);

  console.log('\n--- Summary ---');
  console.log(`Generated: ${successes.length} diagrams`);
  if (failures.length > 0) {
    console.log(`Failed: ${failures.length}`);
    failures.forEach(f => console.log(`  ${f.bundleId}/${f.platform}: ${f.error}`));
  }

  // Verify file sizes (sanity check per Pitfall 6)
  let oversized = 0;
  for (const result of successes) {
    const stats = await fs.stat(result.outputPath);
    const sizeKB = Math.round(stats.size / 1024);
    if (sizeKB > 1024) {
      console.log(`WARNING: ${result.outputPath} is ${sizeKB}KB (>1MB)`);
      oversized++;
    }
  }
  if (oversized > 0) {
    console.log(`${oversized} diagram(s) exceed 1MB — may need optimization`);
  }
}

main().catch(err => { console.error(err); process.exit(1); });
```
