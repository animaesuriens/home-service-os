**`auditYaml(yamlFilename, parsedFlowsData, processesData, bundleDefinitions)` function:**

Takes:
- `yamlFilename` (string): e.g., "boolean-marketing-integration-export.yml"
- `parsedFlowsData` (object): The full parsed-flows.json content
- `processesData` (object): The full processes.json content
- `bundleDefinitions` (array): The definitions array from bundle-definitions.json

Returns an audit result object with this schema:
```javascript
{
  yamlFile: yamlFilename,
  auditedAt: new Date().toISOString(),
  summary: {
    totalFlows: Number,           // total flows in this YAML (business + infra)
    businessFlows: Number,        // non-infrastructure flows
    infrastructureFlows: Number,  // filtered infra flows
    processesMatched: Number,     // processes that have sourceFiles including this YAML
    bundlesCovered: Number,       // bundles that include processes from this YAML
    bundlesTotal: Number,         // total bundle definitions
    unmappedFlows: Number,        // business flows not part of any process/bundle
    accuracyScore: Number         // 0-100: (stepsWithEvidence / totalIdealizedSteps) * 100
  },
  flowMapping: [                  // one entry per business flow in this YAML
    {
      flowName: String,
      flowNumber: String|null,
      isSubFlow: Boolean,
      stepCount: Number,
      processId: String|null,     // matched P-XX or null
      processName: String|null,
      bundleId: String|null,      // which bundle contains this process, or null
      bundleTitle: String|null,
      status: 'mapped' | 'unmapped' | 'infrastructure'
    }
  ],
  hallucinations: [               // one entry per bundle that covers this YAML
    {
      bundleId: String,
      bundleTitle: String,
      idealizedStepCount: Number,
      analysis: [                 // one entry per idealized step
        {
          stepLabel: String,
          actor: String,
          evidence: 'found' | 'not-found' | 'partial',
          matchedFlowActions: [String],  // actual flow step names that support this
          notes: String           // explanation if partial or not-found
        }
      ],
      stepsFound: Number,
      stepsNotFound: Number,
      stepsPartial: Number,
      stepAccuracy: Number        // percentage: found / total * 100
    }
  ],
  missedProcesses: [              // flows in this YAML not covered by any bundle
    {
      flowName: String,
      flowNumber: String|null,
      stepCount: Number,
      description: String,
      keyActions: [String],       // extracted from flow steps (componentKey values)
      showcasePotential: 'high' | 'medium' | 'low'  // heuristic: high if stepCount >= 5 and has multiple componentKeys
    }
  ]
}
```

**Hallucination detection logic:**
For each bundle whose processIds include processes from this YAML:
1. Collect all actual flow steps from the YAML flows that belong to those processes
2. For each idealizedStep in the bundle definition:
   a. Search the actual flow steps for evidence (case-insensitive substring matching of the step label keywords against flow step names and componentKeys)
   b. Mark as 'found' if a clear match exists (step name contains key action words from the label)
   c. Mark as 'partial' if related but not exact (e.g., component exists but action differs)
   d. Mark as 'not-found' if no supporting evidence in any flow step
3. Populate `matchedFlowActions` with the actual step names that matched
4. Populate `notes` with reasoning for partial/not-found (e.g., "No lead scoring component found in any flow step")

**Showcase potential heuristic for missed processes:**
- `high`: stepCount >= 5 AND has >= 2 unique componentKeys (non-infrastructure)
- `medium`: stepCount >= 3 OR has >= 2 unique componentKeys
- `low`: everything else

**`formatAuditMarkdown(auditResult)` function:**

Takes the audit result object and returns a Markdown string with these sections:

```markdown
# Audit: {yamlFilename}
Generated: {auditedAt}

## Summary
- Total flows: {totalFlows} ({businessFlows} business, {infrastructureFlows} infrastructure)
- Processes matched: {processesMatched}
- Bundles covered: {bundlesCovered} / {bundlesTotal}
- Unmapped flows: {unmappedFlows}
- Overall accuracy: {accuracyScore}%

## Flow Mapping

| Flow | # | Steps | Process | Bundle | Status |
|------|---|-------|---------|--------|--------|
{for each flowMapping entry, one row}

## Hallucination Analysis

### {bundleTitle} ({bundleId})
Accuracy: {stepAccuracy}% ({stepsFound} found, {stepsPartial} partial, {stepsNotFound} not found)

| Step | Actor | Evidence | Matched Actions | Notes |
|------|-------|----------|-----------------|-------|
{for each analysis entry, one row}

## Missed Processes (Not in Any Bundle)

| Flow | Steps | Key Actions | Showcase Potential |
|------|-------|-------------|-------------------|
{for each missedProcesses entry, one row}

## Recommendations
- {auto-generated recommendations based on findings}
```

Recommendations logic:
- If stepsNotFound > 0 for any bundle: "Bundle '{bundleTitle}' has {N} hallucinated steps that should be removed or replaced with verified actions"
- If missedProcesses with high showcasePotential: "Flow '{flowName}' is a strong candidate for a new bundle ({stepCount} steps, {keyActions})"
- If unmappedFlows > 0: "{N} flows are not covered by any bundle"

1. Create `linkedin-posts/_pipeline/scripts/audit-yaml.js`:

```javascript
'use strict';

const fs = require('fs-extra');
const path = require('path');
const { auditYaml, formatAuditMarkdown } = require('../lib/audit-engine');
const { loadBundleDefinitions } = require('../lib/bundle-curator');
const { getYamlFileList } = require('../lib/yaml-loader');

function parseArgs() {
  const args = process.argv.slice(2);
  let yamlFilter = null;
  for (let i = 0; i < args.length; i++) {
    if (args[i] === '--yaml' && args[i + 1]) {
      yamlFilter = args[i + 1];
      i++;
    }
    if (args[i] === '--help' || args[i] === '-h') {
      console.log('Usage: node audit-yaml.js --yaml <filename>');
      console.log('');
      console.log('Audits a single YAML file against bundle definitions.');
      console.log('Produces JSON and Markdown audit reports in data/audits/');
      console.log('');
      console.log('Options:');
      console.log('  --yaml <filename>   YAML file to audit (REQUIRED)');
      console.log('  --help, -h          Show this help message');
      console.log('');
      console.log('Valid YAML files:');
      getYamlFileList().forEach(f => console.log('  - ' + f));
      process.exit(0);
    }
  }
  return { yamlFilter };
}

async function main() {
  const { yamlFilter } = parseArgs();

  if (!yamlFilter) {
    console.error('ERROR: --yaml <filename> is required');
    console.error('Run with --help to see valid YAML files');
    process.exit(1);
  }

  // Validate yaml filename
  const validFiles = getYamlFileList();
  if (!validFiles.includes(yamlFilter)) {
    console.error(`ERROR: Unknown YAML file: ${yamlFilter}`);
    console.error('Valid files:');
    validFiles.forEach(f => console.error('  - ' + f));
    process.exit(1);
  }

  const dataDir = path.join(__dirname, '..', 'data');
  const auditsDir = path.join(dataDir, 'audits');

  // Load prerequisite data
  console.log('Loading pipeline data...');
  const parsedFlowsData = await fs.readJSON(path.join(dataDir, 'parsed-flows.json'));
  const processesData = await fs.readJSON(path.join(dataDir, 'processes.json'));
  const bundleDefinitions = loadBundleDefinitions();

  console.log(`Auditing: ${yamlFilter}`);
  console.log(`Against ${bundleDefinitions.length} bundle definitions and ${processesData.totalProcesses} processes\n`);

  // Run audit
  const auditResult = auditYaml(yamlFilter, parsedFlowsData, processesData, bundleDefinitions);

  // Determine output filenames based on YAML index
  const yamlIndex = validFiles.indexOf(yamlFilter) + 1;
  const jsonPath = path.join(auditsDir, `yaml-${yamlIndex}-audit.json`);
  const mdPath = path.join(auditsDir, `yaml-${yamlIndex}-audit.md`);

  // Write outputs
  await fs.ensureDir(auditsDir);
  await fs.writeJSON(jsonPath, auditResult, { spaces: 2 });
  console.log(`JSON audit written to: ${jsonPath}`);

  const markdown = formatAuditMarkdown(auditResult);
  await fs.writeFile(mdPath, markdown, 'utf-8');
  console.log(`Markdown audit written to: ${mdPath}`);

  // Print summary
  console.log('\n=== Audit Summary ===');
  console.log(`Flows: ${auditResult.summary.businessFlows} business, ${auditResult.summary.infrastructureFlows} infrastructure`);
  console.log(`Bundles covered: ${auditResult.summary.bundlesCovered} / ${auditResult.summary.bundlesTotal}`);
  console.log(`Unmapped flows: ${auditResult.summary.unmappedFlows}`);
  console.log(`Accuracy score: ${auditResult.summary.accuracyScore}%`);

  if (auditResult.missedProcesses.length > 0) {
    const highPotential = auditResult.missedProcesses.filter(m => m.showcasePotential === 'high');
    console.log(`\nMissed processes: ${auditResult.missedProcesses.length} (${highPotential.length} high showcase potential)`);
  }
}

main().catch(err => {
  console.error('Audit failed:', err);
  process.exit(1);
});
```
