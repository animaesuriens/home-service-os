**Here's how n8n makes this happen:**

When new contact data flows in from any source, the automation first overrides the contact payload with the incoming information. Then it runs a smart double-search — first looking for an existing contact by their unique ID in HubSpot, then cross-referencing by email address to catch any records that might have been created differently.

The system intelligently selects which search result to use, then merges the incoming data with whatever existing contact information it finds. Based on whether a contact already exists or not, n8n automatically routes the workflow to either update the existing record or create a brand new one.

The beautiful part? It returns the final contact details immediately, so any downstream automations in your workflow have clean, accurate data to work with.
